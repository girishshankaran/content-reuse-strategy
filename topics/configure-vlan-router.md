# Configure VLAN on a router

Use this procedure to create a VLAN interface on the router.

## Steps

1. Open **Configuration > Routing > VLANs**.
2. Click **Add VLAN**.
3. Enter the VLAN ID and interface details.
4. Save and deploy the configuration.

## Verification

Confirm that the new VLAN interface is listed in the device status view.
